# Scientific-Topic-Clustering-Trend-Analysis-NLP-
This project analyzes 150,000 arXiv abstracts (2023–2025) to identify and track current scientific trends. Using unsupervised learning (TF‑IDF, PCA, K‑Means), it produces 12 coherent research clusters, visualizes topic relationships and reveals fast‑growing areas. The results help a company make data‑driven decisions about academic collaboration.
